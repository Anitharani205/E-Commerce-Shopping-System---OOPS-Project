 Order Management System (Java Console Application)

Project Description:
The Order Management System is a simple Java console-based application that allows users to place orders, view their order details, and cancel orders.
This project demonstrates core Java programming concepts such as classes, objects, collections, and basic control flow.

 Features
 Place a new order
 Automatically generate Order ID
 Display user order details
 Show total amount and ordered items
 Cancel an existing order
 Console-based user interaction

Technologies Used
Language: Java
IDE: Eclipse / IntelliJ IDEA / VS Code
Java Version: JDK 8 or above
